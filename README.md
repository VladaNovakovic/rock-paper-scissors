Simple game built to test Angular 2 and webpack using Angular2 starter [https://github.com/preboot/angular-webpack](https://github.com/preboot/angular-webpack)

### Quick start

```bash
# clone our repo
$ git clone https://github.com/preboot/angular-webpack.git my-app

# change directory to your app
$ cd my-app

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```
go to [http://localhost:8080](http://localhost:8080) in your browser.
